# To-Do List Web Application

A clean and functional web application built to create and manage a dynamic list of daily tasks.

## 🚀 Features
- *Dynamic Task Addition:* Capture user input using JavaScript and add tasks instantly to the list.
- *Task Removal Feature:* Implemented a delete button for every task that dynamically removes the elements from the webpage.
- *Input Validation:* Safe handling of empty tasks to prevent adding blank items to the list.
- *Custom DOM Structure:* Dynamically creates task wrapper containers (div), text blocks (span), and control elements (button) on the fly.

## 🛠️ Tech Stack
- *HTML5:* Basic application layout structure.
- *CSS3:* Custom styles for clean user interface elements, buttons, and layouts.
- *JavaScript (Vanilla):* Direct DOM manipulation, custom functions, element creation, and event-driven interaction.

## 📱 JavaScript Concepts Used
- document.getElementById() for selecting inputs and containers.
- document.createElement() to dynamically spawn HTML elements.
- .appendChild() to update and render new list elements.
- .remove() method to eliminate elements from the DOM tree.
